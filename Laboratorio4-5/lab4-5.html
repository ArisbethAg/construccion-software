Preguntas laboratorio 4
¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Ambos son lenguajes de programación que pueden ser implementados en diversos dispositivos y que son muy usados actulamente sin embargo hay algunas diferencias entre ellos como:
-Java es un lenguaje compilado y Javascript uno interpretado.
-Java es un lenguaje orientado a objetos y Javascript es un lenguaje basado en prototipos.
-Java es un lenguaje fuertemente tipado y Javascript es un lenguaje débilmente tipado.
(Isdicoders, 2017; Cerón. M., s.f.)

¿Cómo se declara una variable con alcance local dentro de una función?
Para que una variable tenga un alcance local dentro de un entorno, en este caso una función, se tiene que declarar con let.
¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
Debido a que las variables globales pueden ser accesadas y modificadas por cualquiera mientras el programa esté en ejecución eso hace que nuestras funciones y nuestro programa se vulnere y que pueda tener comportamientos inesperados debido a la misma.
Laborarorio 5
Describe Material Design
Material design es un lenguaje de diseño que fue creado por google con el que se emulan objetos de la vida real y a través del cuál se busca mantener un diseño consistente en las interfaces para los usuarios, así como proveer a los diseñadores de lineamientos para que se puedan crear interfaces que enriquezcan la experiencia del usuario. Algunos de los lineamientos que se establecen en este son:
-El uso de luz y sombra, por ejemplo si hay un botón, esté también tiene qe tener una pequeña sombra, como en la vida real.
-Se debe de procurar usar figuras grandes y colores llamativos en el diseño de interfaces.
-Se debe de buscar incorporar el movimiento en la interfaz, pues es una forma de poder comunicarse con el usuario.
(Enginess, 2016)
