¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

El propósito de los métodos es poder realizar peticiones para realizar acciones en un recurso determinado. De manera más específica este es el propósito de cada uno.

  • GET es el método que es utilizado para leer o recuperar un recurso de la web.
  • HEAD es el método que recupera un recurso de la web, pero sin el cuerpo de la respuesta.
  • POST es el método con el cual se pueden crear nuevos recursos en la web.
  • PUT es el método con el que se actualiza o modifica un recurso en su totalidad.
  • PATCH es el método con el cuál se puede modificar o actualizar una parte de un recurso.
  • DELETE es el método que borra un recurso.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Depende, pues el error 404 quiere decir que se pudo conectar al servidor pero que no pudo encontrar el sitio o que este no existe, si este error sucede debido a que un link se encuentra roto el desarrollador si tendría que arreglarlo, sin embargo si esto tiene que ver más con el dominio y los enlaces del sitio no sería necesario que el desarrollador lo resuelva. (inboundCycle, 2021; Borrás, H., 2021)

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Sí, pues el error 500 quiere decir que posiblemente hay un fallo en la programación, por lo que, quiere decir que el error provino por parte del desarrollador. (Noguera, D., 2020)

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML4 es la versión anterior de HTML5 y sus principales diferencias son que en HTML5 las etiquetas de marcado fueron simplificadas y que, por lo tanto se basa en un estándar nuevo, además de que tiene un análisis sintáctico distinto. (Barrón, B., 2021)

¿Qué componentes de estructura y estilo tiene una tabla?

Una tabla tiene los siguientes componentes de estructura y estilo:

  • Título (<caption>)
  • Filas (<tr>>)
  • Encabezados (<thead>)
  • Encabezados de columnas (<th>)
  • Celdas (<td>)
  • Agrupación de columnas (<colgroup>)
  • Aplicación de atributos y estilos a una o más columnas (<col>)
  • Cuerpo (<tbody>)
  • Sección de pie de tabla (<tfoot>)
